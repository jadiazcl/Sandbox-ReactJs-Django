### Prerequisites

  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Jor-Diaz/Sandbox-ReactJs-Django.git
   ```
2. Execute build Docker Compose
   ```sh
   docker-compose build
   ```
3. Start Docker
   ```sh
   docker-compose up
   ```
